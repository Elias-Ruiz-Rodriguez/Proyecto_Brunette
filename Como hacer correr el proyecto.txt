1- Entrar a la carpeta "ruize"
2- Usar "python manage.py runserver" para hacer funcionar django
3- http://localhost:8000/admin/ para ver donde esta 
4- Activar el entorno virtualenv ./venv/Scripts/activate
5- python -m pip list para ver las versiones
Migrar:
 python manage.py makemigrations
 python manage.py migrate


Como usar git
git branch => consulta de las branch
git checkout -b "nombre" => para crear un nuevo branch
git checkout => cambiarse de branch remota
git fetch --all => Actualizar las branch remotas
git branch -a => verificar las ramas remotas


git add . => para agregar todo lo que agregaste 
git commit "que cambios tuvo"
git push origin "nombre del branch"


git pull origin "nombre del branch" => Actualizar la rama local con los ultimos cambios
